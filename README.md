# 리본 REBORN
전국의 유기동물이 다시(Re) 태어날 수 있는(Born) 기회를 제공하는 앱입니다.

우리 주변에 어떤 유기동물들이 있을까요?

# 주요 기능
* 시, 도별 유기동물 정보 조회
* 동물의 상세 정보 한 눈에 확인하기
* 관심있는 동물들만 모아보기
* 해당 보호소에 연락하기

작은 관심으로 이 친구들에게 리본을 달 수 있도록 해주세요.

리본은 정부 및 해당 동물 보호소들과 무관합니다.
해당 정보의 출처는 농림축산식품부에서 제공하는 동물 보호 관리 시스템의 유기동물 정보 조회 서비스에 있습니다.
